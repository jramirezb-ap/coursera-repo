# Simple Interest Calculator

This repository contains a simple interest calculator written in Bash.  
It computes simple interest using the formula:

Simple Interest = (Principal × Rate × Time) / 100

## Usage
1. Run the script: `./simple-interest.sh`
2. Enter the principal, rate, and time when prompted.
3. The script outputs the calculated simple interest.

## Example
Enter principal amount: 1000
Enter rate of interest (in %): 5
Enter time period (in years): 2
Simple Interest is: 100
